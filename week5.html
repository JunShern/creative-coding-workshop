Week 5

Wild wild web

This week, we peek further outside of the p5.js sandbox we’ve been living in, and realize that there’s so much more that we can play with.

We’ll learn how to create and manipulate browser elements like sliders, checkboxes, buttons, text input, even access the webcam on the browser - we'll spend some time on this last element, taking the opportunity to dig into a bit of image processing.

And then, we’re going to create my personal favorite project in this course - the wacky and ingenious Checkbox Mirror (idea borrowed from Daniel Shiffman)!
