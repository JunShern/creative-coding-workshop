The sessions will be catered to absolute beginners, and the focus of these lessons will be building programs for the sake of fun and understanding.

We’ll be learning to program in JavaScript. Quick facts:

  • 🥇 The “most commonly used programming language in the world for the past 6 years” according to the 2018 Stack Overflow developer survey.
  • 🌐 JavaScript powers the interactivity behind what we think of as The Internet.
  • 🕹 JavaScript is a popular choice for all kinds of things: websites, games, desktop + mobile apps, data visualization and machine learning.
  • 🎨 We’ll be making heavy use of p5.js, a JavaScript library which aims to make coding accessible for artists, designers, educators, and beginners.
